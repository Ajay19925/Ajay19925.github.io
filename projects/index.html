Project/Research Works

Below are the Project/Research works which I did in different courses during my master's program

  • Secure Data Deletion for File System by Automated Process

    From 'Operating Systems' Course

    Description: The permanent removal of data from filesystem is a major area of concern mainly because of the misconception that once a file is deleted or storage media is formatted, it cannot be recovered. When a file is deleted, only the file system entry is deleted, thus the content of the file will remain intact for a period of time, depending on the disk activity afterwards. In my Paper I completely analyzed NTFS filesystem and proposed a way to make our file system more secure by making the permanently deleted file unrecoverable after certain amount of time automatically.

    For more information, this paper can be found in this pdf.
